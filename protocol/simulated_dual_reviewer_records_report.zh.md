# 模拟双人评审记录报告

- 53 个 jointly reported strata 模拟复核记录：`tables/simulated_dual_reviewer_strata_adjudication.csv`
- 28 个 analysis-ready pairs 模拟复核记录：`tables/simulated_dual_reviewer_pair_adjudication.csv`
- 模拟一致性汇总：`tables/simulated_dual_reviewer_agreement_summary.csv`

重要边界：这些记录是 algorithmic/simulated duplicate review stress test，不是真实独立人工双人审核；不得在投稿稿中写作 real inter-reviewer agreement 或真实 Cohen's kappa。

模拟结果：

- Strata 模拟原始一致率：96.2%；模拟 κ：0.92
- Pair 模拟原始一致率：96.4%；模拟 κ：0.91
