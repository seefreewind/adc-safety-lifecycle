# AE Extraction Guide

## Extraction Priority

Extract core safety outcomes first:

1. any AE
2. grade 3 or higher AE
3. serious AE
4. fatal AE
5. AE leading to permanent discontinuation
6. AE leading to dose reduction
7. AE leading to dose interruption

Then extract predefined clinical safety concepts from FDA reviews, ClinicalTrials.gov, publications, and supplementary materials.

## Required Source Capture

Each value must include:

- source document ID
- page, table, figure, or section locator
- analysis population
- denominator
- data cutoff date when available
- AE definition, including all-cause versus treatment-related
- grade category
- seriousness category

## Review

The following fields require independent double review:

- safety analysis population
- data cutoff date
- fatal AE count
- serious AE count
- AE-related discontinuation
- key special-interest safety concepts
- first label inclusion date

At least 20% of ordinary AE rows should be double reviewed.
