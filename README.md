# ADC Safety Lifecycle Public Analysis Deposit

This deposit contains public analytic materials for the manuscript:

**Cross-Source Comparability of Aggregate Safety Outcomes in Pivotal Antibody-Drug Conjugate Trials: A Regulatory Evidence Study**

## Contents

- `code/`: scripts used to build analytic tables, comparability summaries, uncertainty estimates, figures, audit outputs, and manuscript drafts.
- `data/tables/`: derived analytic CSV files used for the Drug Safety-oriented manuscript revision.
- `figures/`: generated figure files for the Drug Safety-oriented revision.
- `manuscript/`: manuscript Markdown, generated table text, project checklist, source-citation inventory, and reference lists.
- `protocol/`: cohort definition, extraction guide, comparability rules, audit reports, visual QA reports, and execution status notes.
- `file_manifest.txt`: full file listing for this deposit package.

## Important Boundary

This deposit intentionally excludes copyrighted full-text journal articles, publisher supplementary PDFs, FDA source PDFs, rendered manuscript page images, and any local raw document folders. The included tables are derived analytic outputs and audit records intended to support reproducibility of the reported aggregate analyses.

## Simulated Duplicate Review Records

The files beginning with `simulated_dual_reviewer_` are algorithmic/internal stress-test records only. They are not records of a real independent second human reviewer and must not be cited as real inter-reviewer agreement or real Cohen's kappa.
